# Linux-Reverse-Shell
Linux tcp reverse shell written in C.

Clone the project :
```
git clone https://github.com/Silme94/Linux-Reverse-Shell
cd Linux-Reverse-Shell/bin
```

Use this command to compile the file :
```
make
```

Use netcat to host a server :
```
nc -nlvp 1234
```

Run the output file :
```
./shell.sh
```
